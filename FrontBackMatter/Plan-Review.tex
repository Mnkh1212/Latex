\noindent Энэхүү дипломын ажил нь Монгол улсын зах зээлд тохирсон, хиймэл оюун ухаанд суурилсан хувийн санхүүгийн хяналт болон зөвлөмжийн \textbf{FinTrack} нэгдсэн системийг боловсруулсан. Хэрэглэгч 5 банкны (Хаан, Голомт, Худалдаа Хөгжлийн Банк, Хас, Төрийн) PDF/XLSX/CSV хэлбэрийн хуулгыг автоматаар уншуулж, гүйлгээгээ ангиллын дагуу нэгтгэн харах, сарын төсөв тогтоох, AI чатаар хувийн санхүүгийн зөвлөгөө авах боломжтой. Системийг \emph{4-давхаргат архитектур}-аар бүтээж, сервер талд Go + Gin, хэрэглэгч талд React Native + Expo, өгөгдлийн санд PostgreSQL 16 ба MinIO обьект хадгалалт ашигласан. Газарзүйн хязгаарлалтыг даван туулахын тулд \textbf{олон нийлүүлэгчтэй AI давхарга} (Gemini → OpenRouter автомат нөөц шилжилт)-ийг шинэлэгээр зохион бүтээсэн нь ажлын онцлог юм. Үр дүнд нь нийт \textbf{45 REST API цэг, 11 өгөгдлийн хүснэгт, 17 дэлгэцтэй} ажиллах прототип бүтсэн бөгөөд бодит хуулгаар туршихад банкны формат таних, ангиллын автомат тогтоох, AI хураангуйн чанар хүлээгдэж буй түвшинд хүрсэн байна. Уг ажил нь арга зүйн хувьд бүрэн биелэгдсэн, практик ач холбогдол өндөртэй бөгөөд дипломын хамгаалалтад оруулахыг дэмжиж байна.\\[0.8cm]
